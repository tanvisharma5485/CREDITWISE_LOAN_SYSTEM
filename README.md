# CreditWise Loan Approval Prediction System

## Overview

This project predicts whether a loan application will be approved or not using Machine Learning.

The system analyzes applicant details like:

* Income
* Credit Score
* Loan Amount
* Education
* Employment Status

and predicts the loan approval status.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

1. Install dependencies
